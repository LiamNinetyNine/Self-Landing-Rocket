# Self-Landing-Rocket
Self directed simulated reinforcement learning based project for determining rocket trajectory from open space to landing ground position

Resources used:  
• https://web.stanford.edu/class/psych209/Readings/SuttonBartoIPRLBook2ndEd.pdf  
• https://github.com/openai/gym

Project outcome: produce an efficient machine learning algorithm for the task at hand
Project reasoning: education and production of real world function
